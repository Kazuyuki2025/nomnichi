研究テーマ

テーマ:
必要な計算機資源とユーザ権限を抑えたコンテナ管理ツールの実現
概要:
動作に必要なメモリやストレージといった計算機資源とroot権限といったユーザ権限を抑えたコンテナ管理ツールを実現する研究を行っている． コンテナ管理ツールとは Docker に代表されるコンテナの作成，およびコンテナイメージの管理を行うプログラムである． 既存のコンテナ管理ツールはデーモンによるコンテナの管理といった過剰な計算機資源の利用や，root権限といった特別なユーザ権限の利用が問題となっている． そこで，必要な計算機資源とユーザ権限を抑えたコンテナ管理ツールを実現する．
本研究で作成したコンテナ管理ツール orca は以下の GitHub レポジトリで管理されている．
https://github.com/nomlab/orca
